# CMZ_VIBRO
Разработка системы вибромониторинга



1) ПЕРЕВОД НА SPI
2) ИСПОЛЬЗОВАНИЕ DMA
3) ЦИФРОВАЯ ОБРАБОТКА (core 0)
   3.1 УДАЛЕНИЕ g ФВЧ
   3.2 ИНТЕГРИРОВАНИЕ С ДРЕЙФОМ (? или на сервере как у ZLab)
   3.3 РАСЧЕТ СКЗ
4) ЗАПИСЬ В БУФЕР ФОРМИРОВАНИЕ JSON ОТПРАВКА (core 1)

---------------------
Реализован опрос по SPI
Далее работа DMA
Создание задач FreeRTOS
Обработка
   Удаление постоянной составляющей
   БПФ и интегрирование в частотной области
ОТПРАВКА отсчетов виброскрости пакетами
