# CMZ_VIBRO
Разработка системы вибромониторинга

Данные корректно считываются с датчика и передаются в задачу отправки по сети. 

1. **Сбор данных**  
   Две высокоприоритетные задачи FreeRTOS (по одной на датчик) с помощью аппаратного таймера (период 313 мкс) опрашивают датчики по SPI. Вычисляется модуль вектора ускорения. Отсчёты накапливаются в буфере размером 1024 значения.

2. **Буферизация**  
   Реализован пул из 6 буферов, которые перемещаются по трём очередям:  
   `free` → `data` (заполненные) → `spectrum` (обработанные) → `free`.  
   Это обеспечивает непрерывный поток данных без копирования.

3. **Цифровая обработка (задача `fft_task`)**  
   - Фильтр верхних частот (10 Гц) для удаления постоянной составляющей.  
   - Взвешивание окном Ханна.  
   - БПФ (radix-2, 1024 точки) с использованием оптимизированной библиотеки ESP‑DSP.  
   - Интегрирование в частотной области: `V(f) = A(f) / (2πf)`.  
   - Расчёт СКЗ виброскорости:  


4. **Управление по сигналу «Цикл»**  
   Внешний сигнал на GPIO34 (прерывание по изменению уровня) включает/выключает сбор данных. В простое оборудование не создаёт сетевого трафика.

5. **Передача данных**  
   Задача `websocket_send_spectrum_task` отправляет обработанный буфер в бинарном виде. После отправки буфер возвращается в пул свободных.

| Поле | Тип | Размер, байт | Описание |
| :--- | :--- | :--- | :--- |
| `sensor_id` | `uint32_t` | 4 | Айди датчика (1 ИЛИ 2)|
| `rms_speed` | `float` | 4 | СКЗ виброскорости |
| `accel` | `float[1024]` | 4096 | Отсчеты ускорения |
| `spectrum` | `float[512]` | 2048 | Значения спектра виброскорости |
| **ИТОГО** | | **6152 Байта** | каждые 0,33 секунды ≈18,3 кб/с |
